= 新增或移除廣播網域中的連接埠
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
廣播網域會在叢集建立或加入作業期間自動建立。您不需要從廣播網域手動移除連接埠。

如果網路連接埠連線能力已變更、無論是透過實體網路連線或交換器組態、而且網路連接埠屬於不同的廣播網域、請參閱下列主題：

link:repair_port_reachability.html["修復連接埠連線能力"]
