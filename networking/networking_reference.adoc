= 網路管理總覽
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以使用下列資訊來執行基本的儲存網路管理。您可以設定實體和虛擬網路連接埠（VLAN和介面群組）、使用IPv4和IPv6建立生命、管理叢集中的路由和主機解析服務、使用負載平衡來最佳化網路流量、以及使用SNMP監控叢集。

除非另有說明、否則這些程序適用於ONTAP 所有版本的更新版本。

您應在下列情況下使用這些程序：

* 您想要瞭解ONTAP 各種的不完整網路管理功能。
* 您想要使用CLI、而非System Manager。
