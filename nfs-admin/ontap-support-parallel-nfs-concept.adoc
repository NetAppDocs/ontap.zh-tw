= 了解 ONTAP 對平行 NFS 的支持
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
支援平行NFS（pNFS）ONTAP 。pNFS傳輸協定可讓用戶端直接存取分散在叢集多個節點上的一組檔案資料、藉此提升效能。它可協助用戶端找到磁碟區的最佳路徑。
