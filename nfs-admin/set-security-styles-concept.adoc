= 了解 ONTAP NFS FlexVol 磁碟區的安全模式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
安全樣式可在FlexVol 支援樹狀結構（根或資料磁碟區）和qtree上設定。安全樣式可在建立時手動設定、自動繼承或稍後變更。
