= 安全樣式繼承的運作方式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您在建立新FlexVol 的流通量或qtree時未指定安全樣式、它會以不同的方式繼承其安全風格。

安全樣式會以下列方式繼承：

* 此功能會繼承包含SVM的根磁碟區安全樣式。FlexVol
* qtree會繼承其包含FlexVol 的不穩定區的安全樣式。
* 檔案或目錄會繼承其包含FlexVol 的不穩定磁碟區或qtree的安全樣式。
