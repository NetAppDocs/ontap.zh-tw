= 用於管理名稱服務交換器項目的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以透過建立、顯示、修改及刪除名稱服務交換器項目來管理這些項目。

[cols="35,65"]
|===


| 如果您想要... | 使用此命令... 


 a| 
建立名稱服務交換器項目
 a| 
「vserver服務名稱服務ns交換器建立」



 a| 
顯示名稱服務交換器項目
 a| 
「Vserver服務名稱服務n-switchshow」



 a| 
修改名稱服務交換器項目
 a| 
「vserver服務名稱服務n-switchmodify」



 a| 
刪除名稱服務交換器項目
 a| 
「Vserver服務名稱服務ns交換器刪除」

|===
如需詳細資訊、請參閱每個命令的手冊頁。
