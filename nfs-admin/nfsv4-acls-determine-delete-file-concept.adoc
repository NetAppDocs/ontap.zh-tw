= 如何使用NFSv4 ACL來判斷是否可以刪除檔案ONTAP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
為了判斷是否能刪除檔案、ONTAP 它使用檔案的刪除位元組合、以及包含目錄的刪除子位元。如需詳細資訊、請參閱NFS 4.1 RFC 5661。
