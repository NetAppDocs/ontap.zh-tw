= 使用非保留連接埠掛載NFS匯出的用戶端考量
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
儲存系統必須停用「-mount-rootonly」選項、即使使用者以root登入、也必須支援使用非保留連接埠掛載NFS匯出的用戶端。這類用戶端包括HummingBird用戶端和Solaris NFS/IPv6用戶端。

如果啟用「-mount-rootonly」選項、ONTAP 則不允許使用非保留連接埠的NFS用戶端（即數字大於1,023的連接埠）來掛載NFS匯出。
