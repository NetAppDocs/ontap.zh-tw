= 用於管理LDAP組態的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
有特定ONTAP 的功能可用來管理LDAP組態。

[cols="35,65"]
|===


| 如果您想要... | 使用此命令... 


 a| 
建立LDAP組態
 a| 
`vserver services name-service ldap create`



 a| 
顯示LDAP組態
 a| 
`vserver services name-service ldap show`



 a| 
修改LDAP組態
 a| 
`vserver services name-service ldap modify`



 a| 
刪除LDAP組態
 a| 
`vserver services name-service ldap delete`

|===
如需詳細資訊、請參閱每個命令的手冊頁。
