= 用於管理匯出規則的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
管理匯出規則時、會ONTAP 有一些特定的指令。

[cols="35,65"]
|===


| 如果您想要... | 使用此命令... 


 a| 
建立匯出規則
 a| 
「vserver匯出原則規則建立」



 a| 
顯示匯出規則的相關資訊
 a| 
「Vserver匯出原則規則顯示」



 a| 
修改匯出規則
 a| 
「Vserver匯出原則規則修改」



 a| 
刪除匯出規則
 a| 
「Vserver匯出原則規則刪除」

|===
[NOTE]
====
如果您已設定多個符合不同用戶端的相同匯出規則、請務必在管理匯出規則時保持同步。

====
如需詳細資訊、請參閱每個命令的手冊頁。
