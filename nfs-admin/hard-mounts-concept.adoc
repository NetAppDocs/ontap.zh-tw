= 使用硬掛載
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
疑難排解掛載問題時、您必須確定使用正確的掛載類型。NFS支援兩種掛載類型：軟掛載和硬掛載。基於可靠性考量、您只能使用硬體掛載。

您不應該使用軟掛載、尤其是在NFS逾時頻繁的情況下。競爭情況可能是因為這些逾時而發生、可能導致資料毀損。
