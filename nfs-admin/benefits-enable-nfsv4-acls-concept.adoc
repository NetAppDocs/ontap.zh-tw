= 了解為 ONTAP SVM 啟用 NFSv4 ACL 的好處
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
啟用NFSv4 ACL有許多好處。

啟用NFSv4 ACL的好處包括：

* 更精細地控制使用者對檔案和目錄的存取
* 更好的NFS安全性
* 改善與CIFS的互通性
* 移除每位使用者16個群組的NFS限制
