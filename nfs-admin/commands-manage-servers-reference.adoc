= 用於管理NFS伺服器的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
管理ONTAP NFS伺服器時、會有特定的功能不一的指令。

[cols="35,65"]
|===


| 如果您想要... | 使用此命令... 


 a| 
建立NFS伺服器
 a| 
vserver NFS create



 a| 
顯示NFS伺服器
 a| 
「Vserver NFS show」



 a| 
修改NFS伺服器
 a| 
"vserver NFS modify"



 a| 
刪除NFS伺服器
 a| 
"vserver NFS刪除"



 a| 
隱藏NFSv3掛載點下的「.snapshot」目錄清單

[NOTE]
====
即使已啟用此選項、仍允許明確存取「.snapshot」目錄。

==== a| 
「vserver nfs」命令、啟用「-v3-he-snapshot」選項

|===
如需詳細資訊、請參閱每個命令的手冊頁。
