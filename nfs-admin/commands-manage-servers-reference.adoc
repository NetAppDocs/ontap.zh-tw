= 用於管理NFS伺服器的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
管理ONTAP NFS伺服器時、會有特定的功能不一的指令。

[cols="35,65"]
|===


| 如果您想要... | 使用此命令... 


 a| 
建立NFS伺服器
 a| 
`vserver nfs create`



 a| 
顯示NFS伺服器
 a| 
`vserver nfs show`



 a| 
修改NFS伺服器
 a| 
`vserver nfs modify`



 a| 
刪除NFS伺服器
 a| 
`vserver nfs delete`



 a| 
隱藏 `.snapshot` NFSv3 掛載點下的目錄列表

[NOTE]
====
明確存取 `.snapshot` 即使已啟用此選項、仍會允許目錄。

==== a| 
`vserver nfs` 的命令 `-v3-hide-snapshot` 選項已啟用

|===
如需詳細資訊、請參閱每個命令的手冊頁。
