= ONTAP NAS SVM 的名稱對映轉換規則
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
這個系統可為每個SVM保留一組轉換規則。ONTAP每個規則包含兩個部分：_Pattern_和_replace_。轉換從適當清單的開頭開始、並根據第一個相符規則執行替代。模式是UNIX樣式的規則運算式。取代是包含轉義序列的字串、代表模式中的子運算式、如同 UNIX `sed` 方案。
