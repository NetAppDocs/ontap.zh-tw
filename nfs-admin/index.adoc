[role="lead"]
包含NFS傳輸協定可用的檔案存取功能。ONTAP您可以啟用NFS伺服器並匯出Volume或qtree。

您可在下列情況下執行這些程序：

* 您想要瞭解ONTAP 各種不同的NFS傳輸協定功能。
* 您想要執行較不常見的組態和維護工作、而非基本NFS組態。
* 您想要使用命令列介面（CLI）、而非System Manager或自動化指令碼工具。
