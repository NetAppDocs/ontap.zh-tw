= 儲存傳輸協定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
有些客戶選擇將整個工作負載移至雲端。這可能比單純使用雲端來保護資料更為複雜。但是由於您不需要重新寫入應用程式、就能使用雲端型儲存設備、所以這個功能讓移動變得更簡單。ONTAP在雲端上執行的功能就像內部部署的功能一樣。ONTAP ONTAP
