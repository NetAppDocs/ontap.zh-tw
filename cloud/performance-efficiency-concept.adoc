= 雲端效能與效率總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您的內部部署ONTAP 的支援功能可提供資料效率功能、讓您儲存更多資料、減少實體空間、並分層極少使用的資料、以降低儲存成本。無論您是使用混合雲組態、或是將整個工作負載移至雲端、ONTAP 都能讓您將儲存效能與效率最大化。
