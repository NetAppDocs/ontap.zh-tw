= 儲存VM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
儲存VM（SVM）可為用戶端和主機提供資料服務。SVM就像虛擬機器在Hypervisor上執行一樣、是將實體資源抽象化的邏輯實體。

在內部ONTAP 部署的不支援環境中、您可以使用SVM來分隔工作負載。在支援的範例Cloud Volumes ONTAP 中、您可以使用多個SVM、也可以使用Cloud Volumes ONTAP 多個執行個體。
