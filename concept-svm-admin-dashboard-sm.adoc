= 了解 ONTAP System Manager SVM 管理員儀表板
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
從 ONTAP 9.19.1 開始，SVM 管理員可以使用專用的 System Manager 儀表板和「儲存 > 磁碟區」頁面來監控和管理其 SVM。這在租用戶型 ONTAP 環境中尤其有用，叢集管理員可以為每個租用戶提供一個 SVM 儀表板。

叢集管理員向 SVM 管理員提供登入憑證。SVM 管理員可以在 SVM 控制台和「儲存 > 磁碟區」頁面查看詳細資訊和效能。控制面板顯示 SVM 協定、網路介面、詳細資訊和效能指標。「磁碟區」頁面提供有關 SVM 磁碟區的資訊，並且（如果叢集管理員允許）SVM 管理員可以新增磁碟區。
