= 啟動或停止節點總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
基於維護或疑難排解的理由、您可能需要啟動或停止節點。您可以從ONTAP 支援的CLI、開機環境提示字元或SP CLI執行此動作。

使用SP CLI命令「系統關機」或「系統關機後再開機」來關閉節點或重新開機、可能會導致節點關機不正常（也稱為「_styled shutshut__」）、也無法取代使用ONTAP 「系統節點停止」命令正常關機。
