= 以每週AutoSupport 更新訊息傳送的檔案
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
每週AutoSupport 更新訊息包含額外的組態和狀態資料、有助於追蹤系統隨時間變化。

以下資訊會以每週AutoSupport 的資訊傳送：

* 每個子系統的基本資訊
* 所選「/mroot/etc/」目錄檔案的內容
* 記錄檔
* 提供系統資訊的命令輸出
* 其他資訊、包括複寫資料庫（RDB）資訊、服務統計資料等
