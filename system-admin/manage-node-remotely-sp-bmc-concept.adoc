= 使用SP/BMC總覽從遠端管理節點
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用內建的控制器（稱為服務處理器（SP）或基礎板管理控制器（BMC）、從遠端管理節點。此遠端管理控制器包含在所有目前的平台機型中。無論節點的作業狀態為何、控制器都能維持正常運作。

下列平台支援BMC而非SP：

* 資訊8700 FAS
* 部分FAS
* FAS27x0
* 解答800 AFF
* S4A700s AFF
* 解答400 AFF
* 解答320 AFF
* VA220 AFF
* C190 AFF
