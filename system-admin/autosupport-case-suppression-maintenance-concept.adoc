= 排程維護期間的個案抑制AutoSupport
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
利用抑制支援的案例功能、您可以避免不必要的案例被列入排程維護時段的訊息所觸發。AutoSupport AutoSupport

若要抑制AutoSupport 不景、您必須以AutoSupport 特殊格式的文字字串「maint=xh」手動叫用不景的訊息。「x」是指維護時段的持續時間、單位為小時。
