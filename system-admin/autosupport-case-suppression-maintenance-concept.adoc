= 排程維護期間的個案抑制AutoSupport
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
利用抑制支援的案例功能、您可以避免不必要的案例被列入排程維護時段的訊息所觸發。AutoSupport AutoSupport

若要隱藏 AutoSupport 個案、您必須手動呼叫具有特殊格式文字字串的 AutoSupport 訊息： `MAINT=xh`。 `x` 為維護期間的持續時間、以小時為單位。
