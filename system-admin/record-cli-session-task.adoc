= 記錄CLI工作階段
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「系統指令碼啟動」和「系統指令碼停止」命令來記錄CLI工作階段。

.步驟
. 若要開始將目前的CLI工作階段錄製到檔案中、請使用「系統指令碼start」命令。
+
如需使用「系統指令碼start」命令的詳細資訊、請參閱手冊頁。

+
將CLI工作階段記錄到指定的檔案中。ONTAP

. 繼續進行CLI工作階段。
. 若要停止錄製工作階段、請使用「系統指令碼停止」命令。
+
如需使用「系統指令碼停止」命令的詳細資訊、請參閱手冊頁。

+
停止記錄您的CLI工作階段。ONTAP
