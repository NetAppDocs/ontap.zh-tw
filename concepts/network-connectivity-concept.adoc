= 網路架構總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
實作資料中心的網路架構ONTAP 通常包含叢集互連、叢集管理的管理網路、以及資料網路。NIC（網路介面卡）提供實體連接埠以供乙太網路連線使用。HBA（主機匯流排介面卡）提供實體連接埠以供FC連線使用。
