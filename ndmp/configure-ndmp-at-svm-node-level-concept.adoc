= 在SVM層級或節點層級總覽設定NDMP
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果備份應用程式支援叢集感知備份（Cab）、您可以在叢集（管理SVM）層級將NDMP設定為SVM範圍、以便備份叢集不同節點上託管的所有磁碟區。否則、您可以設定節點範圍的NDMP、以便備份該節點上裝載的所有磁碟區。
