= 設定SVM範圍的NDMP總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果DMA支援叢集感知備份（Cab）擴充、您可以啟用SVM範圍的NDMP、設定備份使用者帳戶、以及設定資料與控制連線的LIF、來備份叢集中不同節點上託管的所有磁碟區。

DMA必須支援駕駛擴充功能。
