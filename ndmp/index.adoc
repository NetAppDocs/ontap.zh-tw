= NDMP組態總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以快速設定ONTAP 一個叢集、使用網路資料管理傳輸協定（NDMP）、使用協力廠商備份應用程式將資料直接備份到磁帶。

在下列情況下設定NDMP：

* 叢集執行ONTAP 的是功能不正常的9。
* 您擁有第三方備份應用程式（也稱為資料管理應用程式或DMA）。
* 您是叢集管理員。
* 您想要在叢集層級（使用管理儲存虛擬機器（SVM））或節點層級執行備份作業。
* 安裝磁帶設備和可選的媒體服務器。
* 磁帶設備透過光纖通道（FC）交換器連接至叢集、而非直接連接。
* 至少有一個磁帶設備的邏輯單元號碼（LUN）為0。
