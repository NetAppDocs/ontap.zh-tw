= ONTAP SVM 遷移後手動切換客戶端
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
根據預設、在SVM移轉達到「準備轉換」狀態之後、會自動執行用戶端轉換至目的地叢集。如果您選擇停用自動用戶端轉換、則需要手動執行用戶端轉換。

.步驟
. 手動執行用戶端轉換：
+
[source, cli]
----
vserver migrate cutover -vserver <SVM_name>
----
. 檢查轉換作業的狀態：
+
[source, cli]
----
vserver migrate show
----
