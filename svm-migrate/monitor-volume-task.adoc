= 監控ONTAP卷遷移
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
除了使用監控 SVM 整體移轉之外 `vserver migrate show` 命令、您可以監控 SVM 所包含磁碟區的移轉狀態。

.步驟
. 在目標群集上，檢查卷宗遷移狀態：
+
[source, cli]
----
vserver migrate show-volume
----
