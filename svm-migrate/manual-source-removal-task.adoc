= 在用戶端轉換之後手動移除來源SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您在停用來源清理的情況下執行SVM移轉、則可在用戶端轉換完成後手動移除來源SVM。

.步驟
. 確認他們的狀態已準備好進行來源清理：
+
「Dest_cluster」> vserver移轉show

. 清除來源：
+
「Dest_cluster」> vserver會移轉來源清理-vserver <vserver_name>
