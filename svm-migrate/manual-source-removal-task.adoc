= 客戶端切換後手動移除來源ONTAP SVM
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您在停用來源清理的情況下執行SVM移轉、則可在用戶端轉換完成後手動移除來源SVM。

.步驟
. 確認狀態為已準備好進行來源清理：
+
[source, cli]
----
vserver migrate show
----
. 清除來源：
+
[source, cli]
----
vserver migrate source-cleanup -vserver <SVM_name>
----
