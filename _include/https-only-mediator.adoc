* 安裝或升級至 ONTAP Mediator 1.12 或更高版本時、您可以選擇安裝同時支援 iSCSI 和 HTTPS 的 ONTAP Mediator、或僅安裝支援 HTTPS 的 ONTAP Mediator。如果您選擇僅安裝支援 HTTPS 的 ONTAP Mediator、則不會安裝 SCST 套件。
* 如果您從安裝了 SCST 的 ONTAP Mediator 1.11 或更早版本升級，並且在升級過程中選擇僅 HTTPS 選項，則 SCST 軟體包將被卸載，新的 ONTAP Mediator 版本將不包含 SCST 軟體包。
* 如果您打算使用 ONTAP Mediator 監控多個 MetroCluster IP 組態，支援取決於您的 Mediator 類型、ONTAP 版本和作業系統。請參閱 link:https://docs.netapp.com/us-en/ontap-metrocluster/install-ip/concept_mediator_requirements.html#mediator-multiple-mccip["使用 ONTAP Mediator 監控多個 MetroCluster IP 組態"^] 以取得更多資訊。
* 僅支援 HTTPS 的 ONTAP Mediator 安裝方式適用於下列使用情境：
+
** 您計劃僅使用 ONTAP Mediator 來監控您的 SnapMirror active sync 叢集。
** 您計劃設定 ONTAP Mediator 與 HTTPS 以監控一個或多個 MetroCluster IP 組態。


* 在下列情況下，必須安裝支援 iSCSI 和 HTTPS 的 ONTAP Mediator：
+
** 您計劃設定 ONTAP Mediator 與 iSCSI 來監控一個或多個 MetroCluster IP 組態。
