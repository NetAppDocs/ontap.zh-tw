= SVM災難恢復工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要從災難中恢復並從目的地SVM提供資料、您必須啟動目的地SVM。啟動目的地SVM包括停止排程的SnapMirror傳輸、中止進行中的SnapMirror傳輸、打破複寫關係、停止來源SVM、以及啟動目的地SVM。
