= 管理快照保留總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
_snapshot reserve 會預留一定百分比的磁碟空間供快照使用，預設為 5% 。由於快照在快照保留空間用盡時會佔用作用中檔案系統的空間，因此您可能需要視需要增加快照保留空間。或者，您也可以在保留空間已滿時自動刪除快照。
