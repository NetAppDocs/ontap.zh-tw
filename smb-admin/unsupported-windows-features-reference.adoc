= ONTAP SMB 中不支援的 Windows 功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在您的網路中使用CIFS之前、您必須注意ONTAP 到某些不支援的Windows功能。

不支援下列Windows功能：ONTAP

* 加密檔案系統（EFS）
* 在變更日誌中記錄NT檔案系統（NTFS）事件
* Microsoft檔案複寫服務（FRS）
* Microsoft Windows索引服務
* 透過階層式儲存管理（HSM）進行遠端儲存
* 從Windows用戶端進行配額管理
* Windows配額語義
* LMHOSTS.檔案
* NTFS原生壓縮
