= 了解如何將 Microsoft 管理主控台與本機 ONTAP SMB 使用者和群組結合使用
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從Microsoft管理主控台檢視本機使用者和群組的相關資訊。有了這個版本ONTAP 的功能、您就無法從Microsoft管理主控台為本機使用者和群組執行其他管理工作。
