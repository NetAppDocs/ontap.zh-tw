= NTLM驗證
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
以密碼為基礎、根據使用者專屬密碼的共享知識、使用挑戰回應傳輸協定來完成NTLM用戶端驗證。

如果使用者使用本機Windows使用者帳戶建立SMB連線、則CIFS伺服器會使用NTLMv2在本機執行驗證。
