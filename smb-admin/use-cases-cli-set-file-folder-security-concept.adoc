= 使用CLI設定檔案和資料夾安全性的使用案例
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
由於您可以在本機套用及管理檔案與資料夾安全性、而無需遠端用戶端介入、因此您可以大幅縮短設定大量檔案或資料夾的大量安全性所需的時間。

在下列使用案例中、您可以使用CLI設定檔案和資料夾的安全性：

* 在大型企業環境中儲存檔案、例如在主目錄中儲存檔案
* 資料移轉
* Windows網域變更
* 跨NTFS檔案系統的檔案安全性與稽核原則標準化
