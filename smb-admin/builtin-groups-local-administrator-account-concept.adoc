= 使用BUILTIN群組和本機系統管理員帳戶的準則
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您使用BUILTIN群組和本機系統管理員帳戶時、請謹記以下幾項準則。例如、您可以重新命名本機系統管理員帳戶、但無法刪除此帳戶。

* 系統管理員帳戶可以重新命名、但無法刪除。
* 系統管理員帳戶無法從BUILTIN\Administrator群組中移除。
* 可以重新命名內建群組、但無法刪除。
+
在重新命名BUILTIN群組之後、可以使用已知名稱建立另一個本機物件、但會指派新的RID給該物件。

* 沒有本機來賓帳戶。
