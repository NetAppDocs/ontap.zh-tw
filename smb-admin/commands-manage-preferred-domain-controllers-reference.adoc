= 管理慣用網域控制器的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您需要知道用於新增、顯示及移除慣用網域控制器的命令。

|===
| 如果您想要... | 使用此命令... 


 a| 
新增慣用的網域控制器
 a| 
「Vserver CIFS網域preferred DC add」



 a| 
顯示慣用的網域控制器
 a| 
「Vserver CIFS網域慣用DC show」



 a| 
移除慣用的網域控制器
 a| 
「Vserver CIFS網域慣用-DC移除」

|===
