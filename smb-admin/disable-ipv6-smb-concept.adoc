= 停用SMB的IPv6
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
即使使用網路選項在叢集上啟用IPv6、您仍無法使用相同的命令來停用SMB的IPv6。而ONTAP 當叢集管理員停用叢集上最後啟用IPv6的介面時、則會停用IPv6。您應該與叢集管理員溝通、瞭解如何管理啟用IPv6的介面。

如需在叢集上停用IPv6的詳細資訊、請參閱網路管理指南_。
