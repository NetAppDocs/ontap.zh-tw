= 設定BranchCache總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在SMB伺服器上使用ONTAP flexf命令 來設定BranchCache。若要實作BranchCache、您也必須在要快取內容的分公司設定用戶端、以及選擇性地設定裝載的快取伺服器。

如果您將BranchCache設定為依共用區啟用快取、則必須在要提供BranchCache快取服務的SMB共用區上啟用BranchCache。
