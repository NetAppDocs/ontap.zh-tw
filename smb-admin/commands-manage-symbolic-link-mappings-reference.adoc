= 用於管理符號連結對應的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
有特定ONTAP 的功能可用來管理符號連結對應。

|===
| 如果您想要... | 使用此命令... 


 a| 
建立符號連結對應
 a| 
《Vserver CIFS symlink create》（vserver CIFS symlink create）



 a| 
顯示符號連結對應的相關資訊
 a| 
「Vserver CIFS symlink show」



 a| 
修改符號連結對應
 a| 
"Vserver CIFS symlink modify"



 a| 
刪除符號連結對應
 a| 
《Vserver CIFS symlink刪除》

|===
如需詳細資訊、請參閱每個命令的手冊頁。
