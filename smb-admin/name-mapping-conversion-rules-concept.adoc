= 名稱對應轉換規則
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
這個系統可為每個SVM保留一組轉換規則。ONTAP每個規則包含兩個部分：_Pattern_和_replace_。轉換從適當清單的開頭開始、並根據第一個相符規則執行替代。模式是UNIX樣式的規則運算式。置換是一個字串、其中包含代表模式子運算式的轉義序列、如同UNIX的「shed」程式一樣。
