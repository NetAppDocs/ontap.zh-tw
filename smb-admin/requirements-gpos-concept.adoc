= 搭配SMB伺服器使用GPO的需求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要在SMB伺服器上使用群組原則物件（GPO）、您的系統必須符合多項需求。

* SMB必須在叢集上獲得授權。
* SMB伺服器必須設定並加入Windows Active Directory網域。
* SMB伺服器管理狀態必須為開啟。
* 必須設定GPO並套用至包含SMB伺服器電腦物件的Windows Active Directory組織單位（OU）。
* 必須在SMB伺服器上啟用GPO支援。
