= 使用IPv6的需求
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在SMB伺ONTAP 服器上使用IPv6之前、您必須先知道哪些版本的支援哪些版本的支援、以及授權需求為何。



== 不含授權需求ONTAP

取得SMB授權時、IPv6不需要特殊授權。



== SMB傳輸協定版本需求

* 對於SVM、ONTAP 支援所有SMB傳輸協定版本上的IPv6。


[NOTE]
====
不支援透過IPv6提供的NetBios名稱服務（NBNS）。

====
