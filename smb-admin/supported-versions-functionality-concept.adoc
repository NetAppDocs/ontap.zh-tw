= 支援的SMB版本與功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
伺服器訊息區（SMB）是Microsoft Windows用戶端和伺服器所使用的遠端檔案共用傳輸協定。在支援的版本包括所有SMB；不過、預設的SMB 1.0支援則取決於您的版本。ONTAP ONTAP您應該確認ONTAP 支援您環境中所需的用戶端和功能的功能。
