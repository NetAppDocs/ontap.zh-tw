= 指定igroup的啟動器WWPN和iSCSI節點名稱
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在建立igroup時指定啟動器的iSCSI節點名稱和WWPN、也可以稍後新增。如果您選擇在建立LUN時指定啟動器iSCSI節點名稱和WWPN、則可視需要於稍後移除。

請依照主機公用程式文件中的指示取得WWPN、並尋找與特定主機相關的iSCSI節點名稱。對於執行ESX軟體的主機、請使用虛擬儲存主控台。
