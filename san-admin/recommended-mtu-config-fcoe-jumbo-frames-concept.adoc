= FCoE巨型框架的建議MTU組態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
對於乙太網路光纖通道（FCoE）、CNA乙太網路介面卡部分的巨型框架應設定為9000 MTU。CNA FCoE介面卡部分的巨型框架應設定為1500 MTU以上。只有在啟動器、目標和所有介入交換器支援並設定為使用巨型框架時、才設定巨型框架。
