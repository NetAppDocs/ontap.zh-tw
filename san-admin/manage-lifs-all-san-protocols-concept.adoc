= 了解適用於 SAN 環境的 ONTAP LIF
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 系統中的 LUN 和命名空間都包含在儲存虛擬機器（SVM）中。網路介面（LIF）允許 SAN 用戶端連接到 SVM 並存取其包含的 LUN 和命名空間中的資料。

* 您應該為每個支援 SAN 的儲存虛擬機器（SVM）配置一個管理 LIF。
* 每個節點可以配置兩個 LIF，一個用於與 FC 一起使用的每個網狀架構，另一個用於 iSCSI 的獨立乙太網路。
* 支援直接連接或使用乙太網路交換器進行連線。您必須為兩種連線類型設定 LIF。
