= 針對您的環境、判斷正確的Volume與LUN組態組合
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
回答幾個有關您環境的基本問題、有助於判斷FlexVol 最適合您環境的支援功能。

.關於這項工作
您可以最佳化LUN和Volume組態、以達到最大的儲存使用率、或是保證寫入安全。根據您對儲存使用率的需求、以及您快速監控及補充可用空間的能力、您必須判斷FlexVol 適合您安裝的適用的版本。

[NOTE]
====
您不需要為每個LUN分別建立磁碟區。
