= iSCSI的TCP連接埠
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
iSCSI傳輸協定在ONTAP 中設定為使用TCP連接埠編號3260。

不支援變更iSCSI的連接埠號碼。ONTAP連接埠編號3260已登錄為iSCSI規格的一部分、不得用於任何其他應用程式或服務。
