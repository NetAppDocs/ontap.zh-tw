= 檢視叢集儀表板的效能
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
使用儀表板、針對您可能想要新增或移動的工作負載做出明智決策。您也可以查看尖峰使用時間、以規劃可能的變更。

效能值每3秒重新整理一次、效能圖表每15秒重新整理一次。

.步驟
. 按一下*儀表板*。
. 在「*效能*」下、選取時間間隔。
