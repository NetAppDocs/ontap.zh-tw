== 配額總覽

配額提供一種方法來限制或追蹤使用者、群組或qtree所使用的磁碟空間和檔案數目。配額會套用至特定的Volume或qtree。

您可以使用配額來追蹤及限制磁碟區中的資源使用量、並在資源使用量達到特定層級時提供通知。

配額可以是軟的或硬的。軟配額會導致ONTAP 當超過指定的限制時、不再傳送通知、硬配額則會在超過指定的限制時、防止寫入作業成功。
