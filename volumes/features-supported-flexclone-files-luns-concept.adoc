= FlexClone 檔案和 FlexClone LUN 支援的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone檔案和FlexClone LUN可搭配不同ONTAP 的功能、例如重複資料刪除、Snapshot複本、配額和Volume SnapMirror。

FlexClone 檔案和 FlexClone LUN 支援下列功能：

* 重複資料刪除
* Snapshot複本
* 存取控制清單
* 配額
* FlexClone Volume
* NDMP
* Volume SnapMirror
* 。 `volume move` 命令
* 保留空間
* HA 組態
