= FlexClone 檔案和 FlexClone LUN 支援的功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone 檔案和 FlexClone LUN 可搭配不同的 ONTAP 功能使用，例如重複資料刪除，快照，配額和 Volume SnapMirror 。

FlexClone 檔案和 FlexClone LUN 支援下列功能：

* 重複資料刪除
* 快照
* 存取控制清單
* 配額
* FlexClone Volume
* NDMP
* Volume SnapMirror
* 。 `volume move` 命令
* 保留空間
* HA 組態
