= 注意事項：增加FlexVol 適用於整個過程的最大目錄大小
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
預設的最大FlexVol 目錄大小取決於機型、並針對系統記憶體大小進行最佳化。在增加目錄大小上限之前、請先尋求技術支援。

您可以FlexVol 使用「volume modify」命令的「-maildir-sizes」選項來增加特定支援區的預設最大目錄大小、但這樣做可能會影響系統效能。
