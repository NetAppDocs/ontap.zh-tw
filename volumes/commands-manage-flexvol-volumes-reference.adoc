= 管理FlexVol 功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
有一些特定命令可FlexVol 用來使用ONTAP 指令介面來管理各個方面的資料。

[cols="2*"]
|===
| 如果您想要... | 使用此命令... 


 a| 
將Volume上線
 a| 
Volume online（線上Volume）



 a| 
變更Volume的大小
 a| 
磁碟區大小



 a| 
判斷磁碟區的相關Aggregate
 a| 
「Volume show」



 a| 
判斷儲存虛擬機器（SVM）上所有磁碟區的相關Aggregate
 a| 
「Volume show -vserver -Fields Aggregate」



 a| 
判斷磁碟區的格式
 a| 
「Volume show -Fields block-type」



 a| 
使用交會將磁碟區掛載到另一個磁碟區
 a| 
磁碟區掛載



 a| 
將磁碟區置於受限狀態
 a| 
Volume Restrict（Volume限制）



 a| 
重新命名Volume
 a| 
磁碟區重新命名



 a| 
使Volume離線
 a| 
Volume離線

|===
如需詳細資訊、請參閱每個命令的手冊頁。
