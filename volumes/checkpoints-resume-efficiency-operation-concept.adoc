= 使用檢查點來恢復效率作業
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
檢查點用於內部記錄效率作業的執行程序。當效率作業因任何原因（例如系統停止、系統中斷、重新開機、或上次效率作業失敗或停止）而停止、且檢查點資料存在時、效率作業可從最新的檢查點檔案恢復。

建立檢查點：

* 在作業的每個階段或子階段
* 當您執行時 `sis stop` 命令
* 當期間到期時
