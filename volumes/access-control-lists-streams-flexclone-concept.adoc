= 存取控制清單如何搭配FlexClone檔案和FlexClone LUN使用
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
FlexClone檔案和FlexClone LUN會繼承其父檔案和LUN的存取控制清單。

如果父檔案包含Windows NT串流、FlexClone檔案也會繼承串流資訊。但是、無法複製包含六個以上串流的父檔案。
