= qtree名稱限制
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
qtree名稱長度不得超過64個字元。此外、在qtree名稱中使用一些特殊字元（例如：逗號和空格）、可能會導致其他功能發生問題、因此應避免使用。
