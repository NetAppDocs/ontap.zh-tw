= 了解 ONTAP HA 組態如何與 FlexClone 檔案和 FlexClone LUN 搭配運作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
HA組態支援FlexClone檔案與FlexClone LUN作業。

在HA配對中、您無法在進行接管或恢復作業時、在合作夥伴上建立FlexClone檔案或FlexClone LUN。在接管或恢復作業完成後、會恢復合作夥伴上所有擱置的區塊共用作業。
