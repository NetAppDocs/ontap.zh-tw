= 檢查Volume Eff效率 模式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用AFF 「Volume效率show」命令來檢查是否已設定效率、以及檢視目前的效率模式。

.步驟
. 檢查磁碟區的效率模式：
+
`volume efficiency show -vserver <vserver name> -volume <volume name> -fields storage-efficiency-mode`
