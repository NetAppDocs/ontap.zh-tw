= 檢視配額資訊的總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用配額報告來檢視詳細資料、例如配額規則和原則的組態、強制和設定的配額、以及配額調整和重新初始化期間發生的錯誤。

檢視配額資訊在下列情況下非常實用：

* 設定配額、例如設定配額和驗證組態
* 回應即將達到磁碟空間或檔案限制或已達到限制的通知
* 回應更多空間的要求
