= 配額如何搭配qtree運作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用qtree作為目標來建立配額、這些配額稱為_tree配額。您也可以為特定qtree建立使用者和群組配額。此外FlexVol 、有時該磁碟區所包含的qtree會繼承一個適用於該磁碟區的配額。
