= 配額如何與使用者和群組搭配運作的總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以將使用者或群組指定為配額的目標。定義配額時、需要考量幾個實作差異。

您需要注意的一些差異包括：

* 使用者或群組
* UNIX 或 Windows
* 特殊使用者和群組
* 包含多個 ID


根據您的環境、也有不同的方法可以指定使用者 ID 。
