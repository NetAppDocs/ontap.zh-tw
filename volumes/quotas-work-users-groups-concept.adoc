= 配額如何與使用者和群組搭配運作總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您將使用者或群組指定為配額目標時、該配額所設定的限制會套用至該使用者或群組。不過、有些特殊群組和使用者的處理方式卻不同。根據您的環境、有不同的方法可以指定使用者ID。
