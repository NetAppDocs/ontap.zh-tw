= 網路連接埠要求
:allow-uri-read: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
SnapMirror 主動同步複寫對網路連接埠有多個要求。

例如，參與複寫的每個節點都需要監控其夥伴節點的健全狀況、控制複寫活動，而且資料本身也必須複寫。此外，ONTAP Mediator 或 ONTAP Cloud Mediator 需要與所有節點通訊，以便提供仲裁服務並協助容錯移轉。所有這些功能對端點和連接埠的要求各不相同。

在大多數情況下，SnapMirror 主動同步複寫將在內部網路上運作，不受連接埠或目的地限制。如果 IP 流量受到防火牆、路由器、網路拓撲或其他機制的限制，則應驗證下列連線需求。
