= 用於顯示FPolicy組態資訊的命令
:allow-uri-read: 


[role="lead"]
您可以使用「fpolicy show」命令來顯示FPolicy組態的相關資訊、包括FPolicy外部引擎、事件、範圍和原則的相關資訊。

[cols="40,60"]
|===


| 如果您要顯示FPolicy的相關資訊... | 使用此命令... 


 a| 
外部引擎
 a| 
「vserver fpolicy policy exter-engine show」



 a| 
活動
 a| 
「vserver fpolicy事件秀」



 a| 
範圍
 a| 
「vserver fpolicy policy範圍顯示」



 a| 
原則
 a| 
「Vserver fpolicy show」

|===
如需詳細資訊、請參閱命令的手冊頁。
