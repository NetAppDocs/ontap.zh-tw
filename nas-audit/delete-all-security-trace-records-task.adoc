= 刪除所有安全追蹤記錄
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您不想保留任何現有的安全性追蹤記錄、可以使用單一命令刪除節點上的所有記錄。

.步驟
. 刪除所有安全追蹤記錄：
+
「vserver安全性追蹤追蹤追蹤結果刪除-node_name -vserver vserver_name *」

+
** 節點的「node_name」是您要刪除之權限追蹤事件發生的叢集節點名稱。
** vserver的「vserver_name」是您要刪除之權限追蹤事件所在的儲存虛擬機器（SVM）名稱。
