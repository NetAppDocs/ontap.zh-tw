= 瞭解 ONTAP 稽核事件記錄的支援格式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
已轉換的稽核事件記錄檔支援的檔案格式為 `EVTX` 和 `XML` 檔案格式。

您可以在建立稽核組態時指定檔案格式的類型。根據預設、 ONTAP 會將二進位記錄轉換成 `EVTX` 檔案格式。
