= 支援的稽核事件記錄格式為何
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
轉換稽核事件記錄的支援檔案格式為「evtx」和「XML」檔案格式。

您可以在建立稽核組態時指定檔案格式的類型。根據預設ONTAP 、將二進位記錄轉換成「evtx」檔案格式。
