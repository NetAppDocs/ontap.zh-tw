= 存取類型會檢查監控的安全追蹤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
檔案或資料夾的存取檢查是根據多個準則進行。安全追蹤會監控所有這些準則的作業。

安全追蹤監控的存取檢查類型包括：

* Volume與qtree安全樣式
* 檔案系統的有效安全性、其中包含要求執行作業的檔案和資料夾
* 使用者對應
* 共用層級權限
* 匯出層級權限
* 檔案層級權限
* 儲存層級的存取保護安全性
