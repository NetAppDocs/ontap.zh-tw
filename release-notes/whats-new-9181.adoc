== 網路

[cols="30%,70%"]
|===
| 更新 | 說明 


 a| 
xref:../networking/ipsec-prepare.html[IPsec 硬體卸載 IPv6 支持]
 a| 
IPsec硬體卸載支援已擴展到IPv6。



 a| 
xref:../networking/configure_network_security_using_federal_information_processing_standards_fips.html[OpenSSL PQC 演算法]
 a| 
ONTAP支援用於 SSL 的後量子計算加密演算法。這些演算法可針對未來潛在的量子運算攻擊提供額外保護，並且在 SSL FIPS 模式停用時可用。



 a| 
xref:../networking/configure-backend-encryption.html[ONTAP後端叢集網路加密]
 a| 
您可以為ONTAP後端叢集網路上的傳輸中資料啟用 TLS 加密。



 a| 
xref:../high-availability/ha-pair-encryption.html[ONTAP HA 流量網路加密]
 a| 
您可以為高可用性 (HA) 對中的節點之間的流量啟用加密。

|===


== SAN

[cols="30%,70%"]
|===
| 更新 | 說明 


 a| 
xref:../san-admin/nvme-copy-offload.html[NVMe 複製卸載]
 a| 
NVMe 複製卸載功能可讓 NVMe 主機將複製作業從其 CPU 解除安裝至ONTAP儲存控制器的 CPU。主機可以將資料從一個 NVMe 命名空間複製到另一個 NVMe 命名空間，同時將其 CPU 資源保留給應用程式工作負載。

|===

== 安全性

[cols="30%,70%"]
|===
| 更新 | 說明 


 a| 
xref:../anti-ransomware/use-cases-restrictions-concept.html#supported-configurations[FlexGroup磁碟區支援 ARP/AI]
 a| 
NetApp FlexGroup磁碟區支援 ARP/AI，提供基於機器學習的進階勒索軟體偵測和保護。在ONTAP 9.17.1 及更早版本中， FlexGroup磁碟區僅支援第一代 ARP。從ONTAP 9.18.1 開始， FlexGroup磁碟區在本機ONTAP平台（AFF和FAS）和虛擬ONTAP部署（包括Cloud Volumes ONTAP和ONTAP Select ）上與FlexVol磁碟區具有同等功能。因此，ARP/AI 成為FlexVol和FlexGroup磁碟區的預設 ARP 技術。

|===
