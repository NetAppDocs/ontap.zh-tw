= ONTAP 9 版本支援
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從 ONTAP 9.8 版本開始、 NetApp 每個日曆年度提供 ONTAP 版本兩次。雖然計畫可能會有所變更、但其目的是在每個日曆年度的第二季和第四季推出新的 ONTAP 版本。使用此資訊規劃升級的時間範圍、以充分利用最新的 ONTAP 版本。
