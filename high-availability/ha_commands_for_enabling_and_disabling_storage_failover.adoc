= 用於啟用和停用儲存容錯移轉的命令
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
使用下列命令來啟用和停用儲存容錯移轉功能。

|===


| 如果您想要... | 使用此命令... 


| 啟用接管 | `storage failover modify -enabled true -node _nodename_` 


| 停用接管 | `storage failover modify -enabled false -node _nodename_` 
|===

NOTE: 您只能在維護程序中視需要停用儲存容錯移轉。
