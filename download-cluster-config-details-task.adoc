= 下載叢集組態的詳細資料
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
從版本支援的版本9.11.1開始ONTAP 、您可以下載叢集組態的詳細資料。

.步驟
. 在System Manager中、移至*叢集> Overview *。
. 按一下 image:icon-more-kebab-blue-bg.gif["更多選項圖示"] 以顯示動作清單。
. 按一下*下載組態*
. 選取HA配對。
. 按一下*下載*。
