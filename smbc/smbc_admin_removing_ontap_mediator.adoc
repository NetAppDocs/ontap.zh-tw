= 移除ONTAP 此資訊器
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
如果您想要從ONTAP 您的叢集移除現有的支援中心組態ONTAP 、可以使用「撤銷鏡射中介器」命令來執行此作業。

.步驟
. 移除ONTAP 此資訊器：
+
「napmirror中介器移除-nerider-address 12.345.678.90 -peer-cluster cluster cluster cluster _xyz」
