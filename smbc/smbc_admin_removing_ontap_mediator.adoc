= 移除ONTAP 此資訊器
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
如果您想要從 ONTAP 叢集移除現有的 ONTAP Mediator 組態、可以使用來移除 `snapmirror mediator remove` 命令。

.步驟
. 移除ONTAP 此資訊器：
+
`snapmirror mediator remove -mediator-address 12.345.678.90 -peer-cluster cluster_xyz`
