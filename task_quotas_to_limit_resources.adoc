= 設定配額以限制資源使用

:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs:

[role="lead"]
新增配額以限制配額目標可以使用的磁碟空間量。

您可以為配額設定硬限制和軟限制。

硬額度會對系統資源造成硬限制、導致超出限制的任何作業都會失敗。軟配額會在資源使用量達到特定層級時傳送警告訊息、但不會影響資料存取作業、因此您可以在超出配額之前採取適當的行動。

.步驟
. 按一下「*儲存設備>配額*」。
. 按一下「 * 新增 * 」。
