= 將資料提升至效能層級總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從VMware 9.8開始ONTAP 、如果您是進階權限層級的叢集管理員、您可以結合「分層原則」和「雲端擷取原則」設定、從雲端層主動將資料提升至效能層。



== 關於這項工作

如果您想要停止在FabricPool 磁碟區上使用VMware、或是有「僅使用Snapshot」的分層原則、而且想要將還原的Snapshot複製資料恢復到效能層、您可以這麼做。
