= 使用使用者建立的自訂標記總覽進行物件標記
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從使用者建立的自訂標記開始ONTAP 、FabricPool 支援物件標記功能、讓您能夠分類及排序物件、以便更輕鬆地管理。如果您是具有管理權限層級的使用者、可以建立新的物件標記、以及修改、刪除和檢視現有標記。
