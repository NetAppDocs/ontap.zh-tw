= 觸發排定的移轉和分層
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在不想等待預設分層掃描的情況下、隨時觸發分層掃描要求。

.步驟
. 使用帶有「觸發器」選項的「Volume object-store」命令來要求移轉和分層。
+
[listing]
----
volume object-store tiering trigger [ -vserver <vserver name> ] *VServer Name [-volume] <volume name> *Volume Name
----
