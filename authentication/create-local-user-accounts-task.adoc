= 瞭解如何啟用本機 ONTAP 帳戶存取
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
本機帳戶是指帳戶資訊、公開金鑰或安全性憑證位於儲存系統上的帳戶。您可以使用 `security login create`命令啟用本機帳戶來存取管理或資料 SVM 。如link:https://docs.netapp.com/us-en/ontap-cli/security-login-create.html["指令參考資料ONTAP"^]需詳細 `security login create`資訊，請參閱。
