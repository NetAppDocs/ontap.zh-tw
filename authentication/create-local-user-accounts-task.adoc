= 啟用本機帳戶存取總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
本機帳戶是指帳戶資訊、公開金鑰或安全性憑證位於儲存系統上的帳戶。您可以使用 `security login create` 命令以啟用本機帳戶存取管理或資料 SVM 。
