= 管理系統管理員帳戶總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
視啟用帳戶存取的方式而定、您可能需要將公開金鑰與本機帳戶建立關聯、安裝CA簽署的伺服器數位憑證、或設定AD、LDAP或NIS存取。您可以在啟用帳戶存取之前或之後執行所有這些工作。
