= 瞭解 ONTAP S3 儲存區和物件儲存區伺服器原則
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
使用者和群組對S3資源的存取權是由儲存區和物件存放區伺服器原則所控制。如果您的使用者或群組數量不多、在庫位層級控制存取可能就足夠了、但如果您有許多使用者和群組、則更容易控制物件庫伺服器層級的存取。
