== 物件作業

從支援物件中繼資料和標記的ONTAP 支援範圍起、ONTAP 從S9.9開始。

* PutObject 和 CreateMultpartUpload 包含使用的金鑰值配對 `x-amz-meta-<key>.`
+
例如： `x-amz-meta-project: ontap_s3`。

* GetObject：和 HeadObject 會傳回使用者定義的中繼資料。
* 不同於中繼資料、標記可以使用下列方式獨立讀取物件：
+
** PuttObjectTagging
** GetObjectTagging
** 刪除ObjectTagging
