= 在從 ONTAP 9.19.1 回溯之前、請移除第二個 S3 存取金鑰
:allow-uri-read: 


[role="lead"]
如果您升級到 ONTAP 9.19.1 並為任何使用者新增了第二個 S3 存取金鑰，則必須先刪除這些金鑰，才能還原到早期 ONTAP 版本。

.步驟
. link:../s3-config/regenerate-access-keys.html#show-s3-access-keys["顯示 S3 存取金鑰"] 以確定是否已為任何 S3 使用者新增第二個存取金鑰。
. link:../s3-config/regenerate-access-keys.html#remove-an-s3-access-key["移除第二個存取金鑰"] 適用於所有 S3 使用者。
