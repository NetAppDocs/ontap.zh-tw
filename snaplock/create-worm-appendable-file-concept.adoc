= 建立WORM可應用檔案
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
WORM可應用檔案會保留以遞增方式寫入的資料、例如記錄項目。您可以使用任何適當的命令或程式來建立WORM可應用檔案、或是使用SnapLock 「以次磁碟區附加模式_」功能來建立WORM可應用檔案。
