= 瞭解 ONTAP 升級
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您升級 ONTAP 軟體時、您可以利用新的增強 ONTAP 功能、協助您降低成本、加速關鍵工作負載、改善安全性、並擴大組織可用的資料保護範圍。

主要 ONTAP 升級是指從較低版本的 ONTAP 編號版本升級到較高版本的 ONTAP 編號版本。例如，將叢集從 ONTAP 9.14.1 升級到 ONTAP 9.18.1。次要（或修補程式）升級是指在同一編號版本內，從較低版本的 ONTAP 升級到較高版本的 ONTAP。例如，將叢集從 ONTAP 9.17.1P1 升級到 9.17.1P4。

首先，您可以使用下列其中一種方法來準備升級。

* 升級顧問
+
如果您擁有有效的 SupportEdge Active IQ Digital Advisor（也稱為 Digital Advisor）合約，則應該link:create-upgrade-plan.html["準備使用 Upgrade Advisor 升級"]。Upgrade Advisor 提供智慧分析功能，透過評估您的叢集並建立針對您組態的升級計劃，幫助您最大限度地降低不確定性和風險。

+
Upgrade Advisor 則適用於環境規模較小、偏好基於雲端、以使用者介面為中心的使用者體驗的使用者。

+
了解更多有關 link:https://docs.netapp.com/us-en/active-iq/upgrade_advisor_overview.html["升級顧問"^] 的資訊。

* 升級健全狀況檢查程式
+
您也可以link:create-upgrade-plan.html["使用 Upgrade Health Checker 準備升級"]。升級健康檢查器是一款升級諮詢服務工具，可進行現場全面健康檢查，以識別升級障礙和警告。使用升級健康檢查器無需簽訂 SupportEdge 合約。

+
Upgrade Health Checker 適用於擁有大型或複雜環境、網路存取受限或已有自動化和編排框架的使用者。

+
了解更多有關 link:https://docs.netapp.com/us-en/upgrade-health-checker/learn-about-upgrade-health-checker.html["升級健全狀況檢查程式"^] 的資訊。

* 手動準備
+
如果您不使用 Upgrade Advisor 或 Upgrade Health Checker 工具，則需要link:prepare.html["準備手動升級"]。手動準備需要完成多項任務，以檢查任何已知問題或障礙，並確保您的叢集符合升級的必要要求。

+
了解更多有關 link:prepare.html["手動升級準備"^] 的資訊。



準備升級之後，建議您使用執行升級link:task_upgrade_andu_sm.html["從 System Manager 自動進行不中斷升級（ andu ）"]。ANDU 利用 ONTAP 的高可用度（ HA ）容錯移轉技術，確保叢集在升級期間能持續提供資料，而不會中斷。


NOTE: 從ONTAP 9.12.1 開始，系統管理器與NetApp控制台完全整合。如果您的系統上配置了控制台，您可以透過系統頁面進行升級。

如果您需要升級 ONTAP 軟體的協助、 NetApp 專業服務提供 link:https://www.netapp.com/pdf.html?item=/media/8144-sd-managed-upgrade-service.pdf["託管升級服務"^]。如果您有興趣使用此服務、請聯絡您的 NetApp 銷售代表或 link:https://www.netapp.com/forms/sales-contact/["提交 NetApp 的銷售查詢表"^]。託管升級服務以及其他類型的升級支援、均提供給客戶 link:https://www.netapp.com/pdf.html?item=/media/8845-supportedge-expert-service.pdf["SupportEdge Expert 服務"^] 無需額外成本。
