= 可存取服務處理器的使用者帳戶變更
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您在 ONTAP 9.8 和更早版本中建立了使用者帳戶、以便以非系統管理角色存取服務處理器（ SP ）、並升級至 ONTAP 9.9.1 或更新版本、則中的任何非系統管理值都會是如此 `-role` 參數已修改為 `admin`。

如需詳細資訊、請參閱 link:../system-admin/accounts-access-sp-concept.html["可存取SP的帳戶"]。
