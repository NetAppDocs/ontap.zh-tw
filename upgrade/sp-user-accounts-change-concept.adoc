= 可存取服務處理器的使用者帳戶變更
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您在ONTAP 更新版本中建立的使用者帳戶可存取具有非管理員角色的服務處理器（SP）、並升級ONTAP 至版本號為版本號的更新版本、則在「-role」（角色）參數中的任何非管理員值都會修改為「admin」。

如需詳細資訊、請參閱 link:../system-admin/accounts-access-sp-concept.html["可存取SP的帳戶"]。
