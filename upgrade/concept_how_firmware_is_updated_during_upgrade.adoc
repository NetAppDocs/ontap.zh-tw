= 韌體ONTAP 如何在升級過程中更新
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
由於ONTAP 升級功能包括升級韌體、因此您不需要手動更新韌體。當您執行ONTAP 還原升級時、ONTAP 會將包含在升級套件中的叢集韌體複製到每個節點的開機裝置、並自動安裝新的韌體。

如果叢集中的韌體版本比ONTAP 隨附於此版更新套件的韌體版本舊、則會自動更新下列元件的韌體：

* 系統與診斷：
+
** BIOS
** Flash 快取
** 服務處理器（SP）


* 磁碟
* 磁碟櫃


如有需要、您也可以在ONTAP 各個版本之間手動更新韌體。
