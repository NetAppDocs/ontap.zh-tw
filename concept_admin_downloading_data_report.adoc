[role="lead"]
從功能表9.8開始ONTAP 、您可以下載System Manager中顯示的網路相關資料。

當您在_清單檢視_中顯示資訊時、可以按一下*下載*、然後下載顯示的物件清單。

* 此清單會以「以逗號分隔的值（CSV）」格式下載。
* 只會下載可見欄中的資料。
* CSV檔案名稱的格式為物件名稱和時間戳記。
