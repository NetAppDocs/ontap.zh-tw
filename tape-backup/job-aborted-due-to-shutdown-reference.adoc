= 工作因關機而中止
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Job aborted due to shutdown`

* *原因*
+
儲存系統正在重新開機。

* *修正行動*
+
儲存系統重新開機後、請重試工作。
