= 無法鎖定快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Failed to lock snapshot`

* *原因*
+
快照正在使用中或已刪除。

* *修正行動*
+
如果快照正由另一個作業使用，請等待該作業完成，然後重試備份。如果已刪除快照，則無法執行備份。
