= 合格的磁帶機總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必須使用已通過測試且發現可在儲存系統上正常運作的合格磁帶機。您可以追蹤磁帶別名、也可以啟用磁帶保留功能、以確保在任何特定時間只有一個儲存系統存取磁帶機。

合格的磁帶機是已通過測試、可在儲存系統上正常運作的磁帶機。您ONTAP 可以使用磁帶組態檔案、將磁帶機限定為現有的版本。
