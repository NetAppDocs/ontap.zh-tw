= 傳輸失敗（CFO啟動中止）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Transfer failed (CFO initiated abort)`

* *原因*
+
由於CFO Aggregate的儲存容錯移轉（接管和恢復）作業、SMTape作業會中止。

* *修正行動*
+
CFO Aggregate的儲存容錯移轉完成後、請執行SMTape作業。
