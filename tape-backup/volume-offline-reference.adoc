= Volume離線
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「Volume offline」

* *原因*
+
正在備份的Volume已離線。

* *修正行動*
+
將磁碟區上線、然後重試備份。
