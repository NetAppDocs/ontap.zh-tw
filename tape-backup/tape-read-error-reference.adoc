= 磁帶讀取錯誤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
磁帶讀取錯誤

* *原因*
+
磁帶機損壞或磁帶未包含完整備份。

* *修正行動*
+
如果磁帶機損壞、請使用其他磁帶機。如果磁帶未包含完整備份、您將無法還原資料。
