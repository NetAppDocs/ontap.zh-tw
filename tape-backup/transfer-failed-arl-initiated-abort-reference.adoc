= 傳輸失敗（ARL初始中止）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「Transfer failed（ARL初始中止）」（傳輸失敗（ARL初始中止））

* *原因*
+
如果啟動了Aggregate重新定位、則SMTape作業會在進行中、然後中止。

* *修正行動*
+
完成Aggregate重新配置作業後、請執行SMTape作業。
