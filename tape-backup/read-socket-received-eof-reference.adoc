= 讀取套接字接收到EOF
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
"Read Socket received EOF"（讀插槽收到EOF）

* *原因*
+
嘗試以NDMP三向連線與遠端磁帶通訊已達檔案結尾標記。您可能嘗試從區塊大小較大的備份映像進行三方還原。

* *修正行動*
+
請指定正確的區塊大小、然後重試還原作業。
