= 無效的Volume路徑
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`Invalid volume path`

* *原因*
+
找不到用於備份或還原作業的指定磁碟區。

* *修正行動*
+
以有效的Volume路徑和Volume名稱重試工作。
