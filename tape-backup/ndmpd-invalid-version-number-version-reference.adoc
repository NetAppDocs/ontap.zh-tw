= ndmpd無效的版本號碼：VERIN_NUMBER_`
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`ndmpd invalid version number: version_number`

* *原因*
+
儲存系統不支援指定的NDMP版本。

* *修正行動*
+
指定NDMP版本4。
