= 磁帶記錄大小應為block_Size1、而非block_Size2
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`Tape record size should be block_size1 and not block_size2`

* *原因*
+
為本機還原指定的封鎖因素不正確。

* *修正行動*
+
使用重試還原工作 `block_size1` 成為阻礙因素。
