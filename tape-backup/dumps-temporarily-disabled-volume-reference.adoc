= 傾印在磁碟區上暫時停用、請再試一次
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「在Volume上暫時停用、請再試一次」

* *原因*
+
在SnapMirror目的地磁碟區上嘗試進行NDMP傾印備份、此磁碟區是「napmirror中斷」或「napmirror resSync」作業的一部分。

* *修正行動*
+
等待「快照鏡射中斷」或「快照鏡射重新同步」作業完成、然後執行傾印作業。

+
[NOTE]
====
每當SnapMirror目的地Volume的狀態從讀取/寫入變更為唯讀或從唯讀變更為讀取/寫入時、您必須執行基礎備份。

====
