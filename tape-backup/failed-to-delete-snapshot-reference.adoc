= 無法刪除快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「無法刪除快照」

* *原因*
+
無法刪除自動Snapshot複本、因為其他作業正在使用此複本。

* *修正行動*
+
使用「nap」命令來判斷Snapshot複本的狀態。如果不需要Snapshot複本、請手動刪除。
