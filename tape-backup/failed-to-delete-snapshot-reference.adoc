= 無法刪除快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Failed to delete snapshot`

* *原因*
+
無法刪除自動快照，因為其他作業正在使用此快照。

* *修正行動*
+
使用 `snap`命令來判斷快照的狀態。如果不需要快照，請手動刪除。
