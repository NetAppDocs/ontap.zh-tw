= 無法刪除快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Failed to delete snapshot`

* *原因*
+
無法刪除自動Snapshot複本、因為其他作業正在使用此複本。

* *修正行動*
+
使用 `snap` 用於確定 Snapshot 副本狀態的命令。如果不需要Snapshot複本、請手動刪除。
