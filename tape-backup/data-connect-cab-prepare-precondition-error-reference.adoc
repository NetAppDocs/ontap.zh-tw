= 資料連線：CAB資料連線準備先決條件錯誤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`DATA CONNECT: CAB data connection prepare precondition error`

* *原因*
+
當備份應用程式已與NDMP伺服器交涉CAO擴充功能、且NDMP資料連線位址類型與NDMP與NDMP資料連線訊息不符時、NDMP資料連線會失敗。

* *修正行動*
+
請聯絡您的備份應用程式廠商。
