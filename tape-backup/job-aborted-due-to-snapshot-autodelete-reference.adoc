= 由於Snapshot自動刪除、工作已中止
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Job aborted due to Snapshot autodelete`

* *原因*
+
磁碟區空間不足、因此觸發了Snapshot複本的自動刪除。

* *修正行動*
+
釋放磁碟區中的空間、然後重試工作。
