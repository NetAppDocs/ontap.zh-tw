= 無效的輸入磁帶
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「輸入磁帶無效」

* *原因*
+
備份映像的簽名在磁帶標頭中無效。磁帶有毀損的資料或不包含有效的備份映像。

* *修正行動*
+
使用有效的備份映像重試還原工作。
