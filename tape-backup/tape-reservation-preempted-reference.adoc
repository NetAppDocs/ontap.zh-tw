= 磁帶保留已搶佔
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Tape reservation preempted`

* *原因*
+
磁帶機正由其他作業使用、或磁帶已提前關閉。

* *修正行動*
+
請確定磁帶機未被其他作業使用、且DMA應用程式未中止工作、然後重試。
