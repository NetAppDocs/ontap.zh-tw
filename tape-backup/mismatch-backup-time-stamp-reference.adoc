= 備份時間戳記不相符
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`Mismatch in backup time stamp`

* *原因*
+
磁帶變更期間載入的磁帶並非備份集的一部分。

* *修正行動*
+
使用 `smtape restore -h` 用於驗證磁帶標頭資訊的命令。
