= 來自Write Dirnet的訊息：錯誤字串
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Message from Write Dirnet: error_string`

* *原因*
+
以NDMP三向連線將通訊備份到遠端磁帶時發生錯誤。

* *修正行動*
+
檢查遠端移動設備的網路連線。
