= 磁帶讀取時發生媒體錯誤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「磁帶讀取時發生媒體錯誤」

* *原因*
+
正在從中還原資料的磁帶已毀損、可能不包含完整的備份資料。

* *修正行動*
+
如果您確定磁帶有完整備份、請重試還原作業。如果磁帶未包含完整備份、您將無法執行還原作業。
