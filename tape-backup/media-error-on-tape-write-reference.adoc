= 磁帶寫入時發生媒體錯誤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「磁帶寫入時發生媒體錯誤」

* *原因*
+
用於備份的磁帶已毀損。

* *修正行動*
+
更換磁帶並重試備份工作。
