= 傳輸失敗（MetroCluster 因為無法運作而中止）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「Transfer failed（Transfered due to MetroCluster orioperation）」（傳輸失敗（因為無法執行還原作業而中斷）

* *原因*
+
SMTape作業會因切換或切換作業而中止。

* *修正行動*
+
切換或切換作業完成後、請執行SMTape作業。
