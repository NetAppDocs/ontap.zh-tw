= 磁帶寫入失敗-新磁帶已在媒體結尾
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`Tape write failed - new tape is already at the end of media`

* *原因*
+
磁帶空間不足、無法完成備份。

* *修正行動*
+
更換磁帶並重試備份。
