= 當磁碟區存取類型變更時、傾印如何運作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當SnapMirror目的地Volume或SnapVault 還原次要Volume的狀態從讀取/寫入變更為唯讀或從唯讀變更為讀取/寫入時、您必須執行基礎磁帶備份或還原作業。

SnapMirror目的地SnapVault 和不二級磁碟區是唯讀磁碟區。如果您在這類磁碟區上執行磁帶備份與還原作業、則當磁碟區的狀態從唯讀變更為讀取/寫入、或從讀取/寫入變更為唯讀時、就必須執行基礎備份或還原作業。
