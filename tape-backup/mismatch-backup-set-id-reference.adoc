= 備份集ID不相符
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「備份集ID中的相符項目」

* *原因*
+
磁帶變更期間載入的磁帶並非備份集的一部分。

* *修正行動*
+
載入正確的磁帶並重試工作。
