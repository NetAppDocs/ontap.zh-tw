= 管理節點範圍NDMP模式的命令
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「系統服務NDMP」命令、在節點層級管理NDMP。其中有些命令已過時、將在未來的重大版本中移除。

您只能在進階權限層級使用下列NDMP命令：

* 系統服務NDMP服務終止
* 《系統服務NDMP服務啟動》
* 系統服務NDMP服務停止
* 系統服務NDMP記錄啟動
* 系統服務NDMP記錄停止
