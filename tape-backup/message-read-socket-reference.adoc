= 來自讀取套接字的訊息：error_string
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Message from Read Socket: error_string`

* *原因*
+
以NDMP 3向連線從遠端磁帶還原通訊時發生錯誤。

* *修正行動*
+
檢查遠端移動設備的網路連線。
