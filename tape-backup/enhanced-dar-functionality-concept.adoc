= 了解ONTAP NDMP 的增強型 DAR 功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以將增強的直接存取還原（DAR）功能用於檔案和NT串流的目錄DAR和DAR。預設會啟用增強的DAR功能。

啟用增強的DAR功能可能會影響備份效能、因為必須建立偏移對應並寫入磁帶。您可以在節點範圍和儲存虛擬機器（SVM）範圍NDMP模式中啟用或停用增強的DAR。
