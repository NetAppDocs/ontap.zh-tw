= 已在磁帶結尾
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「已經在磁帶的結尾」

* *原因*
+
磁帶不含任何資料、或必須重新繞線。

* *修正行動*
+
如果磁帶不含資料、請使用包含備份的磁帶、然後重試還原工作。否則、請將磁帶倒帶、然後重試還原工作。
