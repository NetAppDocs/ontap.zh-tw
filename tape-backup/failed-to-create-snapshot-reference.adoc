= 無法建立快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`Failed to create snapshot`

* *原因*
+
該磁碟區已包含Snapshot複本的最大數量。

* *修正行動*
+
刪除一些Snapshot複本、然後重試備份作業。
