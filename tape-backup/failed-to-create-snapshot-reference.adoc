= 無法建立快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Failed to create snapshot`

* *原因*
+
磁碟區已包含快照的最大數量。

* *修正行動*
+
刪除一些快照，然後重試備份作業。
