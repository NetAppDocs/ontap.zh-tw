= 磁帶順序不正常
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「磁帶出問題」

* *原因*
+
用於還原作業的磁帶順序第一個磁帶沒有映像標頭。

* *修正行動*
+
將磁帶與映像標頭一起載入、然後重試工作。
