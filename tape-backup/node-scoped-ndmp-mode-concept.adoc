= 什麼是節點範圍的NDMP模式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在節點範圍的NDMP模式中、您可以在節點層級執行磁帶備份與還原作業。從8.2升級到8.3後、將繼續保留在用作執行動作的NDMP模式。Data ONTAP

在節點範圍的NDMP模式中、您可以在擁有該磁碟區的節點上執行磁帶備份與還原作業。若要執行這些作業、您必須在擁有磁碟區或磁帶裝置的節點上、於裝載LIF上建立NDMP控制連線。

[NOTE]
====
此模式已過時、將在未來的主要版本中移除。
