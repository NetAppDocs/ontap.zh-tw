= 找不到快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「找不到快照」

* *原因*
+
指定用於備份的Snapshot複本無法使用。

* *修正行動*
+
檢查指定的Snapshot複本是否可用。如果沒有、請使用正確的Snapshot複本重試。
