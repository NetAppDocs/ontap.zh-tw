= 找不到快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Failed to find snapshot`

* *原因*
+
指定用於備份的快照不可用。

* *修正行動*
+
檢查指定的快照是否可用。如果沒有，請使用正確的快照重試。
