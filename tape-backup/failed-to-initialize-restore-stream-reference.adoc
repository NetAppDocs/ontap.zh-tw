= 無法初始化還原串流
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`Failed to initialize restore stream`

* *原因*
+
您可能會因為下列原因而收到此錯誤訊息：

+
** 備份映像不是SMTape。
** 指定的磁帶封鎖因素不正確。
** 磁帶毀損或損壞。
** 載入錯誤的磁帶以進行還原。


* *修正行動*
+
** 如果備份映像不是SMtape、請使用具有SMtape備份的磁帶重試此作業。
** 如果封鎖因素不正確、請指定正確的封鎖因素、然後重試此作業。
** 如果磁帶毀損、您將無法執行還原作業。
** 如果載入錯誤的磁帶、請使用正確的磁帶重試操作。
