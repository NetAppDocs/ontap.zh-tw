= 無法鎖定傾印所需的快照
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「無法鎖定傾印所需的快照」

* *原因*
+
指定用於備份的Snapshot複本無法使用。

* *修正行動*
+
使用不同的Snapshot複本重試備份。

+
使用「nap list」命令查看可用Snapshot複本的清單。
