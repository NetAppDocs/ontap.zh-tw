= 磁帶寫入失敗-新磁帶毀損或受到寫入保護
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Tape write failed - new tape is broken or write protected`

* *原因*
+
用於備份的磁帶已毀損或受到寫入保護。

* *修正行動*
+
更換磁帶並重試備份。
