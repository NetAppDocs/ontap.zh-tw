= 啟用或停用事件記錄
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以開啟或關閉事件記錄。

.步驟
. 若要啟用或停用事件記錄、請在clusterShell輸入下列命令：
+
hy*選項__選項_名稱_備份.log.enable __選項值_｛開啟|關閉｝*

+
「開啟」會開啟事件記錄。

+
「關」會關閉事件記錄功能。

+
[NOTE]
====
預設會開啟事件記錄。

====
