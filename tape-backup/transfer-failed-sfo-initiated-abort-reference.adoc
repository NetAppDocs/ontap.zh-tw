= 傳輸失敗（ SFO 起始中止）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Transfer failed (SFO initiated abort)`

* *原因*
+
由於儲存容錯移轉（接管與恢復）作業、SMTape作業會中止。

* *修正行動*
+
儲存容錯移轉（接管與恢復）作業完成後、請執行SMTape作業。
