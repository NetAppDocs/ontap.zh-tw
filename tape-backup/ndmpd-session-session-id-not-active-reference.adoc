= ndmpd工作階段工作階段_ID未啟用
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
`ndmpd session session_ID not active`

* *原因*
+
NDMP工作階段可能不存在。

* *修正行動*
+
使用 `ndmpd status` 用於檢視作用中 NDMP 工作階段的命令。
