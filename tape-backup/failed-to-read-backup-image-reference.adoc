= 無法讀取備份映像
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「讀取備份映像失敗」

* *原因*
+
磁帶毀損。

* *修正行動*
+
如果磁帶毀損、您將無法執行還原作業。
