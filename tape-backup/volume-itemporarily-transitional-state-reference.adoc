= Volume暫時處於過渡狀態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Volume is temporarily in a transitional state`

* *原因*
+
要備份的磁碟區暫時處於未掛載狀態。

* *修正行動*
+
等待一段時間、然後再次執行備份。
