= 磁帶寫入失敗-新磁帶發生媒體錯誤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Tape write failed - new tape encountered media error`

* *原因*
+
用於備份的磁帶已毀損。

* *修正行動*
+
更換磁帶並重試備份。
