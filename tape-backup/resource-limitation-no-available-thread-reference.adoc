= 資源限制：無可用的執行緒
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


* *訊息*
+
「資源限制：沒有可用的執行緒」

* *原因*
+
目前使用中的本機磁帶I/O執行緒數目上限。您最多可以有16個作用中的本機磁帶機。

* *修正行動*
+
等待部分磁帶工作完成、再開始新的備份或還原工作。
