= 磁帶記錄大小必須介於4KB與256KB之間
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
* *訊息*
+
`Tape record size must be in the range between 4KB and 256KB`

* *原因*
+
為備份或還原作業指定的封鎖因素不在允許範圍內。

* *修正行動*
+
指定範圍為4 KB至256 KB的封鎖因素。
