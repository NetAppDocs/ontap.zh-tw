= 支援同時運行的ONTAP磁帶設備數量
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在光纖通道、SCSI或SAS附件的任何組合中、每個儲存系統（每個節點）最多可支援64個同步磁帶機連線、16個中型更換器、以及16個橋接器或路由器裝置。ONTAP

磁帶機或媒體交換器可以是實體或虛擬磁帶庫或獨立裝置中的裝置。

[NOTE]
====
雖然儲存系統可以偵測到64個磁帶機連線、但同時執行的最大備份與還原工作階段數量取決於備份引擎的擴充性限制。
