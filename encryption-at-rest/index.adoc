= 了解ONTAP資料加密
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp同時提供軟體與硬體加密技術、確保儲存媒體在重新調整用途、退回、放錯地方或遭竊時、無法讀取閒置的資料。

* 使用NetApp Volume Encryption（NVE）的軟體式加密可一次支援一個磁碟區的資料加密
* 使用NetApp儲存加密（NSE）的硬體式加密可支援寫入資料時的全磁碟加密（FDE）。
