= 使用NVE總覽加密Volume資料
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從使用支援功能9.7開始ONTAP 、如果您擁有VE授權、以及內建或外部金鑰管理、則根據預設會啟用Aggregate和Volume加密。對於支援更新版本的支援、您可以在新磁碟區或現有磁碟區上啟用加密功能。ONTAP您必須先安裝VE授權並啟用金鑰管理、才能啟用Volume加密。NVE符合FIPS-140-2第1級標準。
