= 檢查ONTAP磁碟吞吐量和延遲
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以檢查叢集節點的磁碟處理量和延遲度量、以協助您進行疑難排解。

.關於這項工作
此工作需要進階權限層級命令。

.步驟
. 變更為進階權限層級：
+
[source, cli]
----
set -privilege advanced
----
. 檢查磁碟處理量與延遲度量：
+
[source, cli]
----
statistics disk show -sort-key latency
----


.範例
下列範例顯示每個使用者讀取或寫入作業的總計 `node2` 開啟 `cluster1`：
