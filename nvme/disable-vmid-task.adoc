= 在 ONTAP 中停用 NVMe 主機虛擬機器識別碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從 ONTAP 9.14.1 開始， ONTAP 預設支援 NVMe / FC 主機透過唯一識別碼識別虛擬機器，以及針對 NVMe / FC 主機監控虛擬機器資源使用率的能力。這可強化主機端報告和疑難排解。

您可以使用 bootarg 來停用此功能。請參閱知識庫文章link:https://kb.netapp.com/on-prem/ontap/da/SAN/SAN-KBs/How_to_disable_NVMe_host_virtual_machine_identifier_in_ONTAP["如何在ONTAP中停用 NVMe 主機虛擬機器標識符"^]。
