= 從SnapMirror DR目的地Volume提供資料
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當災難停用SnapMirror DR關係的主要站台時、您可以在不中斷營運的情況下、從目的地磁碟區提供資料。當主要站台還原服務時、您可以重新啟動來源Volume。
