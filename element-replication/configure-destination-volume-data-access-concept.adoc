= 設定目的地Volume以進行資料存取
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
將目的地Volume設為可寫入後、您必須設定該磁碟區以進行資料存取。SAN主機可從目的地磁碟區存取資料、直到重新啟動來源磁碟區為止。

. 將元素LUN對應至適當的啟動器群組。
. 從SAN主機啟動器建立iSCSI工作階段至SAN LIF。
. 在SAN用戶端上、執行儲存設備重新掃描以偵測連接的LUN。
