= Hyper-V和SQL Server在SMB上的不中斷營運意味著什麼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Hyper-V與SQL Server在SMB上的不中斷營運、是指結合各種功能、讓應用程式伺服器與所包含的虛擬機器或資料庫維持線上狀態、並在許多管理工作期間提供持續可用度。這包括儲存基礎架構的計畫性和非計畫性停機。

透過SMB支援的應用程式伺服器不中斷營運包括：

* 計畫性接管與恢復
* 非計畫性接管
* 升級
* 計畫性集合體重新配置（ARL）
* LIF移轉與容錯移轉
* 計畫性Volume搬移
