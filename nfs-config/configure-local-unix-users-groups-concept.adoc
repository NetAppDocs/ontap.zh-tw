= 設定本機UNIX使用者和群組總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用SVM上的本機UNIX使用者和群組進行驗證和名稱對應。您可以手動建立UNIX使用者和群組、也可以從統一資源識別元（URI）載入包含UNIX使用者或群組的檔案。

叢集中的本機UNIX使用者群組和群組成員、預設上限為32、768。叢集管理員可以修改此限制。
