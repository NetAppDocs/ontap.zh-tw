= 判斷哪些磁碟區位於本機層（Aggregate）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可能需要先判斷哪些磁碟區位於本機層（Aggregate）、然後才能在本機層上執行作業、例如重新定位磁碟區或使其離線。

.步驟
. 若要顯示位於Aggregate上的磁碟區、請輸入
+
「Volume show -Aggregate Aggregate _name」

+
將顯示位於指定集合體上的所有Volume。
