= 判斷哪些磁碟區位於集合體上
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可能需要先判斷哪些磁碟區位於集合體上、然後才能對集合體執行作業、例如重新定位或離線。

.步驟
. 若要顯示位於Aggregate上的磁碟區、請輸入
+
「Volume show -Aggregate Aggregate _name」

+
將顯示位於指定集合體上的所有Volume。
