= 鏡射與無鏡射的Aggregate
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用稱為「_SIRS_ SyncMirror 」的選用功能、在儲存在不同RAID群組的複本或_plex_中同步鏡射集合體資料。如果有更多磁碟故障、或是RAID群組磁碟失去連線、則會利用叢碼來確保資料不會遺失。
