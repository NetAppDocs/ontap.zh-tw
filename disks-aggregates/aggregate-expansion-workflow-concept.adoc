= 新增容量至本機層的工作流程（擴充Aggregate）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要新增容量至本機層（擴充Aggregate）、您必須先識別要新增的本機層、判斷需要多少新儲存設備、安裝新磁碟、指派磁碟擁有權、以及視需要建立新的RAID群組。

您可以使用System Manager或CLI來新增容量。
