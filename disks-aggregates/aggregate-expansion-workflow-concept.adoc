= 將容量新增至 ONTAP 中本機層的工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要將容量新增至本機層，您必須先識別要新增的本機層，判斷需要多少新儲存設備，安裝新磁碟，指派磁碟擁有權，並視需要建立新的 RAID 群組。

您可以使用系統管理員或 ONTAP CLI 來新增容量。
