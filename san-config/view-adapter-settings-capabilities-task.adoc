= 檢視介面卡設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要檢視統一化目標介面卡（ X1143A-R6 ）的設定、您必須執行 `system hardware unified-connect show` 顯示控制器上所有模組的命令。

.步驟
. 在不連接纜線的情況下啟動控制器。
. 執行 `system hardware unified-connect show` 命令查看連接埠組態和模組。
. 在設定CNA和連接埠之前、請先檢視連接埠資訊。
