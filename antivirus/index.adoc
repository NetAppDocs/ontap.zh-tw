= 防毒組態總覽
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用NetApp病毒掃描（稱為_VScan）來保護資料、避免受到病毒或其他惡意程式碼的危害。本課程說明當用戶端透過SMB存取檔案時、如何使用存取時掃描來檢查病毒、以及如何使用隨需掃描來立即或排程檢查病毒。

您可以使用ONTAP 不只是System Manager或自動化指令碼工具的功能、還能透過使用功能不全的命令列介面（CLI）來使用VScan。System Manager不支援VScan。
