= 了解如何設定 ONTAP Vscan 掃描器池
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
掃描器集區會定義VScan伺服器和可連線至SVM的授權使用者。掃描器原則會決定掃描器集區是否處於作用中狀態。

[NOTE]
====
如果您在 SMB 伺服器上使用匯出原則、則必須將每個 VScan 伺服器新增至匯出原則。

====
