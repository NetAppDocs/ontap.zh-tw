= 執行每日監控
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以執行每日監控、確保不會發生任何需要注意的立即效能問題。

.步驟
. 從這個功能表、前往*事件目錄*頁面、檢視所有目前和過時的事件。Active IQ Unified Manager
. 從* View（檢視）*選項中、選取「Active Performance Events（作用中效能事件）」、然後決定需要採取什麼行動。
