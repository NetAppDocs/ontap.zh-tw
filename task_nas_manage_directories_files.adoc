展開System Manager Volume（系統管理器）顯示以查看和刪除目錄和檔案。

從版本號為0.9.9.1開始ONTAP 、系統會以低延遲的快速目錄刪除功能刪除目錄。

如需更多關於在ONTAP 更新版本的資訊中檢視檔案系統、請參閱 link:concept_nas_file_system_analytics_overview.html["檔案系統分析總覽"]。

.步驟
. 選擇*儲存>磁碟區*。展開磁碟區以檢視其內容。
